## ✨ 功能特性

### 🎯 核心功能
- **🍅 番茄工作法**：专注（25分钟）、短休息（5分钟）、长休息（15分钟）
- **⏳ 沙漏视觉**：实时显示时间流逝的沙漏动画
- **🎨 双主题模式**：优雅的浅色/深色主题，自动切换
- **🔔 音频提示**：风铃、铃声或静音选项
- **💾 本地存储**：自动保存设置和统计信息

### 🚀 高级功能
- **🖥️ 全屏模式**：沉浸式专注体验
- **📱 响应式设计**：完美适配桌面和移动设备
- **⌨️ 键盘快捷键**：快速操作，无需鼠标
- **⚙️ 自定义设置**：灵活调整各阶段时长
- **📊 统计追踪**：记录每日专注时间和番茄数

## ⚙️ 使用指南

### 1. 开始专注
1. 选择 **🎯 专注** 模式
2. 点击 **▶️ 开始** 按钮（或按空格键）
3. 观察沙漏慢慢变空，专注工作25分钟

### 2. 休息时间
- 专注结束后自动进入 **☕ 短休息**（5分钟）
- 每完成4个番茄，进入 **🌴 长休息**（15分钟）
- 休息期间可以暂停或跳过

### 3. 自定义设置
点击 **⚙️ 设置** 按钮可以：
- 调整各阶段时长（1-60分钟）
- 选择提示音效（🔔 风铃 / 🛎️ 铃声 / 🔇 静音）
- 切换 🌙/☀️ 主题
- 恢复默认设置

### 4. 全屏模式
点击 **⛶ 全屏** 按钮进入沉浸模式，适合深度工作

## ⌨️ 快捷键

| 按键 | 功能 | 说明 |
|------|------|------|
| **空格键** | 开始/暂停 | 切换计时器状态 |
| **R** | 重置 | 重置当前阶段计时 |
| **S** | 跳过 | 跳过当前阶段 |
| **F** | 全屏 | 切换全屏模式 |
| **T** | 主题 | 切换浅色/深色主题 |
| **ESC** | 退出 | 退出全屏/关闭设置 |
